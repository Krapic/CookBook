# CookBook AI

Android aplikacija za otkrivanje i dijeljenje recepata uz pomoć umjetne inteligencije.

## Funkcionalnosti

- Objava recepata
- Pretraga recepata
- Like i rating
- Follow korisnika
- AI import recepta iz teksta
- AI prijedlog recepata iz sastojaka

## Tehnologije

- Kotlin
- Jetpack Compose
- MVVM
- Firebase
- Gemini AI
- Room database
